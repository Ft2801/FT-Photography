# FT-Photography Portfolio

Welcome to the **FT-Photography** repository. This project showcases a professional photography portfolio, highlighting creative works and visual storytelling.

## Project Overview

This is a **Next.js** application designed to display high-quality photography in a sleek, modern interface. It serves as a personal portfolio for Fabio Tempera, demonstrating skills in both photography and web development.

## Live Demo

You can view the live project on GitHub Pages here:
[**FT-Photography Portfolio**](https://ft2801.github.io/FT-Photography/)

## Features

- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Modern UI/UX**: Built with modern web technologies for a smooth user experience.
- **Gallery Showcase**: Displays a collection of selected photographs.
